# Turn Prep Module - Foundry VTT

Add-on module for Foundry VTT that helps players prepare their turn in advance.

## Features

- Plan actions and bonus actions before your turn
- Track reaction triggers
- Save favorite turn plans
- View turn history

## Installation

1. Copy the module to your Foundry modules directory
2. Enable the module in your world settings
3. Open a character sheet and navigate to the "Turn Prep" tab

## Development

```bash
npm install
npm run dev
```
